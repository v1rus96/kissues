Solving Technical Issues

Solving Technical Issues

Technical issues may happen for every student who join to the OTS.

Trainer must be aware about the types of technical issues and their solutions to handle it at the beginning of session.

Here we will talk about most common issues and their solutions.

OTS





Using Zoom Browser

Using Zoom Browser

If you notice that there is no Annotation option while Student is sharing the screen, this indicates that he is using Zoom Web, Chromebook or Tablet.

To solve this problem:

- If the Student using Zoom Web ask him to install the Zoom app
- If the Student using Chromebook or Tablet ask if they have laptop or PC to join with other device.

If they don’t, then check whether Student can proceed without annotations by doing few steps or activities. (DON’T FORGET TO INFORM THE MANAGER OF THE SESSION IN WHATSAPP GROUP)

OTS





Audio Issues

Note

For early ages we may need to have help of parents, so if you see kid cannot understand the guidelines, wait to communicate with their parents from team.

Audio issues: Cannot hear each other

A - Sometimes Student device has audio issues such as:

- Student and trainer cannot hear each other.

- In this case ask the Student to use Chat to communicate with you and ask whether he/she have headphone to connect.

- If it does not work then ask them to check the setting of the Zoom “by clicking on the small Arrow beside the Mic icon and choose different device in Microphone section”.

- Next option would be ask to join with additional device such as Phone or Tablet to communicate with Trainer.

- If you check all and still not working it is probably Zoom issue so can ask Student to rejoin by zoom.

Audio issues: Cannot hear each other

Audio issues: Echo or Background Noise

B - Echo

- If you hear echo, ask the Student or parents to move to separate room to avoid the echo (need to check with manager)


C - Background noise

- If the Student has background noise you can ask the Student or parents to move silent room if it is possible, else ask to connect the headphone.

Camera Issues

Camera Issues

Camera Issues

If the Student have camera issue or it doesn’t work you can ask to join with additional device such as Tabler or Phone.

To avoid echo the device which does not use the mic must reduce the volume to 0 or leave the computer audio from Zoom.

Connection Issues

Connection Issues

Connection issues you may face are:

- Breaking voice
- Camera hanged
- Delay while doing screen sharing
- Disconnections

Solution is to ask the student use Wifi if they are using mobile internet. As the next option, try to proceed without camera; it may reduce network lags. If connection issue cannot be resolved inform the manager.

OTS





Share Screen Issues

Share Screen Issues

Screen sharing issues (On MacOS):

- Student is using Mac which requires a permission

To solve this issue you ask help from parents if the Student is early age and guide them to:

1 - Open System Preferences => Security & Privacy => Privacy Tab => Screen Recording option.

2 - Click on Lock icon to unlock and tick the Zoom option. It will ask to restart the Zoom. Ask parents/Student to ignore it and share the screen.

Share Screen Issues

Share Screen Issues

Screen sharing issues (On Windows):

- This happens rarely but if you face it, ask student/parent to install latest version of the Zoom App and rejoin the same link.

Remote Access Issues

Remote Access Issues

Remote Access Permission:

- MAC devices require permission to do remote access.

To solve this issue you ask help from parents if the Student is early age and guide them to:

1 - Open System Preferences => Security & Privacy => Privacy Tab => Accessibility option.

2 - Click on Lock icon to unlock and tick the Zoom option.

Student CA Link Issues

Student CA Link Issues

CA links issue:

- Links should have specific time according to the session. But if you see the message “Your session haven’t started yet” without specific time, inform manager to set the time.

- Whenever you finish the activity DON’T FORGET to press Next Challenge button to save the code into the system. If you forget to do the code will be lost then you have to go to the link and replicate the website from scratch.

OTS





Using Apple (MacBook) Devices

Notes for Mac Devices

- Student may use trackpad. If it is possible ask them to use a normal mouse.

- Consider that “Right Click” can be 2 fingers tap

- Instead if “CTRL V/C/A” ask the Student to use “Command V/C/A”

Unknown issues

Check everything with your managers and let them to solve the issue.
